## Overview

Critical Frequency is an open-source project dedicated to **transforming the mental health outcomes for the UK's Black Queer Men**.  
Our mission is to create a new community infrastructure that centres Black queer lives—building spaces, tools, and networks that support wellbeing, advocacy, and belonging.

Developed and maintained by [BLKOUTUK](https://github.com/BLKOUTUK), this project strives to enable real, sustainable change and support across communities.

---

## Features

- Community-driven resources and support frameworks
- Tools for mental health assessment and improvement
- Platforms for connection, peer-support, and advocacy
- Extensible architecture for future enhancements
